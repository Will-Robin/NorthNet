NorthNet.Classes.SubstructureNetwork

This code is a model for a 'Substructure-Reaction Rule Network' in which reactive relationships are expressed as functional group interconversions, rather than as explicit reactions.

Reaction rules (NorthNet.Classes.ReactionTemplate objects) connect to substructures (NorthNet.Classes.Substructure objects) as reactant substructures and product substructures. Compounds (NorthNet.Classes.Compound objects) which match substructures connect to them.

Substructures are separate entities so that the 'meaning' of a reaction rule is preserved, even if there are no compounds matching a reactant or product substructure.

It is possible to add and remove compounds, reactions and reaction rules. Adding substructures is meaningless (they are added in the process of adding a reaction or reaction rule). Compounds can only be added if they contain a substructure matching one already in the Substructure Network. There is no restriction on adding reactions and reaction rules.

217        """
218        Add a reaction rule to the network.
219
220        Parameters
221        ----------
222        rule: NorthNet.Classes.ReactionTemplate
223
224        Returns
225        -------
226        None
227        """
228
229        assert isinstance(
230            rule, Classes.ReactionTemplate
231        ), """SubstructureNetwork.add_template:
232            rule arg should be a NorthNet ReactionTemplate object."""

236        if rxn_key not in self.ReactionRules:

244

246
247            working_substruct = self.Substructures[rxn_substr].Mol
248
249            # connect to compounds
250            for comp in self.Compounds:
251                compound = self.Compounds[comp].Mol

255
256        for p_substruct in rule.ProductSubstructures:

261
262            self.Substructures[p_substruct].ProducingReactions.append(rxn_key)

264            working_substruct = self.Substructures[p_substruct].Mol

304    def remove_reaction_rule(self, rule):
305        """
306        Remove a reaction rule from a SubstructureNetwork.
307
308        Parameters
309        ----------
310        rule: NorthNet.Classes.ReactionTemplate
311
312        Returns
313        -------
314        None
315        """
316
317        assert isinstance(
318            rule, Classes.ReactionTemplate
319        ), """SubstructureNetwork.remove_template:
320            template arg should be a NorthNet ReactionTemplate object."""
321
322        # Disconnect the reaction rule from substructures
323        for substr in rule.ReactantSubstructures:
324            substructure = self.Substructures[substr]
325            rxn_partic = substructure.ReactionParticipations
326            if rule.ReactionSMARTS in rxn_partic:
327                rxn_partic.remove(rule.ReactionSMARTS)
328
329        for substr in rule.ReactantSubstructures:
330            substructure = self.Substructures[substr]
331            prod_rxns = substructure.ProducingReactions
332            if rule.ReactionSMARTS in prod_rxns:
333                prod_rxns.remove(rule.ReactionSMARTS)
334
335        # Check for disconnected substructures
336        remove_substructs = []
337        for substr in self.Substructures:
338            substructure = self.Substructures[substr]
339            rxn_partic = substructure.ReactionParticipations
340            prod_rxns = substructure.ProducingReactions
341            if len(prod_rxns) == 0 and len(rxn_partic) == 0:
342                remove_substructs.append(substr)
343
344        # disconnect the substructure from any matching compounds.
345        for r_substr in remove_substructs:
346            for comp in self.Substructures[r_substr].MatchingCompounds:
347                self.Compounds[comp].ReactiveSubstructures.remove(r_substr)
348            del self.Substructures[r_substr]
349
350        # Check for disconnected compounds
351        remove_compounds = []
352        for comp in self.Compounds:
353            compound = self.Compounds[comp]
354            if len(compound.ReactiveSubstructures) == 0:
355                remove_compounds.append(comp)
356
357        for comp in remove_compounds:
358            del self.Compounds[comp]
359
360        # Finally, remove the reaction rule
361        del self.ReactionRules[rule.ReactionSMARTS]

501    def convert_to_networkx(self):
502        """
503        Converts NorthNet network object to networkx object.
504
505        Parameters
506        ----------
507
508        Returns
509        -------
510        G: networkx.DiGraph
511            Networkx version of the NorthNet SNetwork.
512        """
513        from NorthNet.Utils import sha1_hash
514
515        G = nx.DiGraph()
516
517        # create some aliases for the substructures
518        # (they cannot be SMARTS strings) used as node names
519        substructure_aliases = {s: sha1_hash(s) for s in self.Substructures}
520        template_aliases = {s: sha1_hash(s) for s in self.ReactionRules}
521
522        for c in self.Compounds:
523            compound_alias = self.Compounds[c].SMILES
524
525            for i in self.Compounds[c].ReactiveSubstructures:
526                G.add_edge(substructure_aliases[i], compound_alias)
527
528        for r in self.ReactionRules:
529
530            template = self.ReactionRules[r]
531
532            for reac in template.ReactantSubstructures:
533                G.add_edge(substructure_aliases[reac], template_aliases[r])
534
535            for prod in template.ProductSubstructures:
536                G.add_edge(template_aliases[r], substructure_aliases[prod])
537
538        return G
